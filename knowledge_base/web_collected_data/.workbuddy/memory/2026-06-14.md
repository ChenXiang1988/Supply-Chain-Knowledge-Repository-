## 搭建微信文章转 Markdown 工具 (wx2md)

为用户在工作目录下搭建了 `wx2md` 工具，用于将 PC 端微信文章批量抓取并转换为 Markdown 文档。

### 产出文件
- `wx2md/wx2md.py` — 核心脚本（Playwright + markdownify）
- `wx2md/requirements.txt` — Python 依赖
- `wx2md/links.txt` — 文章链接列表模板
- `wx2md/README.md` — 使用说明
- `wx2md/output/` — 生成的 Markdown 输出目录（运行时自动创建）

### 技术要点
- 使用 Playwright 无头浏览器绕过微信反爬
- 图片通过浏览器 fetch + base64 下载，本地化保存到 `images/` 目录
- 输出包含 YAML frontmatter（title/author/date/url/captured_at）
- Python 环境隔离在 `~/.workbuddy/binaries/python/envs/wx2md`

### 使用方式
1. 将微信文章链接贴入 `wx2md/links.txt`（每行一个）
2. 运行：`C:\Users\MartinCX\.workbuddy\binaries\python\envs\wx2md\Scripts\python.exe wx2md/wx2md.py`
3. 结果输出到 `wx2md/output/YYYY-MM-DD-标题/` 目录
