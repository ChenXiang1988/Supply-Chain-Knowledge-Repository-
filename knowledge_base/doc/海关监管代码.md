# 海关进出口监管方式代码

> 说明：这里整理的是日常业务中最常见、确定性较高的海关监管方式代码，便于产品、运营、关务和客服快速对照。  
> 如用于正式申报，请以最新海关口径和报关单填制要求为准。

---

## 常用代码速查

| 代码 | 监管方式 | 解释 |
| --- | --- | --- |
| `0110` | 一般贸易 | 最常见的正常进出口贸易方式，适用于一般买卖成交的货物。 |
| `0130` | 易货贸易 | 直接以出口货物交换进口货物的贸易方式，不涉及货币支付。 |
| `0214` | 来料加工装配贸易 | 外商提供料件，国内企业按约定加工装配后复出口。 |
| `0615` | 进料对口 | 进料加工（对口合同），企业自购料件加工后出口。 |
| `0654` | 进料深加工 | 进料加工项下保税货物深加工结转。 |
| `0420` | 加工贸易设备 | 加工贸易项下进口的设备、模具等。 |
| `4019` | 边境小额贸易 | 边境地区按边贸政策开展的小额进出口。 |
| `9610` | 跨境电商零售出口 | 跨境电子商务零售出口，常见于 B2C 出口场景。 |
| `9710` | 跨境电商企业对企业直接出口 | 跨境电商企业通过线上平台实现的 B2B 直接出口。 |
| `9810` | 跨境电商出口海外仓 | 跨境电商出口至海外仓后，再面向海外终端销售。 |

---

## 业务理解

- `0110` 是最基础的正常贸易口径。
- `0214`、`0615`、`0654` 属于加工贸易相关口径，申报时要和手册、合同、备案信息保持一致。
- `0420`、`4019`、`9610`、`9710`、`9810` 属于常见的专项业务口径，通常需要和真实业务场景严格匹配。
- 保税、减免税类业务，申报前必须按规定完成备案、审批、办理减免税证明等前置手续，保证单证一致。
- 暂时进出境货物需严格遵守海关规定时限，逾期未复运进出境，将按规定补征税费并处置。

## 补充说明

1. **本表汇总企业日常高频使用监管代码，完整代码请以海关总署官方发布的《监管方式代码表》为准。**
2. **保税、减免税类业务**，申报前必须按规定完成备案、审批、办理减免税证明等前置手续，保证单证一致。
3. **跨境电商监管方式**存在地域、试点政策差异，正式申报前务必确认属地海关执行口径。
4. **暂时进出境货物**需严格遵守海关规定时限，逾期未复运进出境，将按规定补征税费并处置。
5. **一份报关单只允许填报一种监管方式**，请根据实际业务场景选择最匹配的代码。
6. 监管方式代码由4位数字构成：前两位为分类代码，后两位为贸易方式代码。
