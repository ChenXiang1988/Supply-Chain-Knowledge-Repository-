《关于完善跨境电子商务零售进口监管有关工作的通知》的个人分析

2018年11月30日，商务部 发展改革委 财政部 海关总署 税务总局 市场监管总局六部委联合下发了《关于完善跨境电子商务零售进口监管有关工作的通知》(商财发〔2018〕486号)，《通知》在海关总署2016年26号公告基础上对开展跨境电子商务的主体进行了修改使主体责任更加明晰并对跨境电商商务监管进行了进一步的规范，现本人对《通知》的一些条款进行个人分析，仅供大家讨论。

一、《通知》第一条“本通知所称跨境电商零售进口，是指中国境内消费者通过跨境电商第三方平台经营者自境外购买商品，并通过“网购保税进口”（海关监管方式代码1210）或“直购进口”（海关监管方式代码9610）运递进境的消费行为。”

此条明确了跨境电商零售进口应该是交易在前，然后电商销售企业再选择“1210”或者“9610”模式，而目前绝大多数城市开展“网购保税进口”模式时，却是电商商品先进入海关特殊监管区域后再进行销售，根据海关总署2014年第57号关于增列海关监管方式代码的公告第一条“增列海关监管方式代码“1210”，全称“保税跨境贸易电子商务”，简称“保税电商”。适用于境内个人或电子商务企业在经海关认可的电子商务平台实现跨境交易，并通过海关特殊监管区域或保税监管场所进出的电子商务零售进出境商品（海关特殊监管区域、保税监管场所与境内区外（场所外）之间通过电子商务平台交易的零售进出口商品不适用该监管方式）。”的规定，括号中的排除条款也指出目前电商商品先进入海关特殊监管区域后再进行销售的模式不属于“1210”范畴，也就是说不属于“网购保税进口”模式。而海关总署2014年第12号关于增列海关监管方式代码的公告第一条“增列海关监管方式代码“9610”，全称“跨境贸易电子商务”，简称“电子商务”，适用于境内个人或电子商务企业通过电子商务交易平台实现交易，并采用“清单核放、汇总申报”模式办理通关手续的电子商务零售进出口商品（通过海关特殊监管区域或保税监管场所一线的电子商务零售进出口商品除外）。”的规定中并未将电商商品先进入海关特殊监管区域后再进行销售的模式排除。我认为没有实现零售交易的时候，通过海关特殊区域“一线”入区的货物是否能称为“跨境电商商品”是值得商榷的，而“网购保税进口”模式是为了方便已经完成零售交易的电商商品在海关特殊区域内分拣，打包，按收件地址集中拼柜，降低企业在境外开展上述工作的成本。

二、《通知》第一条第三款“未通过与海关联网的电子商务交易平台交易，但进出境快件运营人、邮政企业能够接受相关电商企业、支付企业的委托，承诺承担相应法律责任，向海关传输交易、支付等电子信息。”

此款规定为现在快件渠道的代购现象指明了一条合法的道路，为2019年1月1日正式实施的《中华人民共和国电子商务法》做了提前铺垫。

根据海关总署公告2016年第19号关于启用新快件通关系统相关事宜的公告第一条“A类快件是指无商业价值的文件、单证、票据和资料（依照法律、行政法规以及国家有关规定应当予以征税的除外）。B类快件是指境内收寄件人（自然人）收取或者交寄的个人自用物品（旅客分离运输行李物品除外）。C类快件是指价值在5000元人民币（不包括运、保、杂费等）及以下的货物（涉及许可证件管制的，需要办理出口退税、出口收汇或者进口付汇的除外）。”的规定，B类快件只能适用于个人自用物品，而目前大量的海外代购商品都是通过此渠道流入到境内。代购大部分为个人行为，经常为了偷逃税款而伪报瞒报，甚至贩卖假货，扰乱了国家正常经济秩序，海关也无法对此类贸易进行统计。《中华人民共和国电子商务法》的出台，从根本上杜绝了此类现象的发生，《电子商务法》第九条、第十条和第十二条，不论是个人、代购店、跨境交易的网站或其他组织，如果在海外从事代购，必须要在中国的工商局进行登记并领取对应的营业执照。如果在平台注册的卖家没有营业执照，平台将有权报送相关管理部门并予以处罚。国家把代购也纳入到正规企业管理中来，降低了制假贩假的风险，对可能发生的消费纷争，消费者也可以通过正规法律渠道得到解决。

三、《通知》第二条第一款“跨境电商零售进口经营者（以下简称跨境电商企业）：自境外向境内消费者销售跨境电商零售进口商品的境外注册企业，为商品的货权所有人。”

此款是对海关总署2016年第26号公告的修改，把跨境电商企业由境内注册企业修改为境外注册企业。

跨境电子商务的定义是指不同关境的交易主体，通过电子商务平台达成交易、进行支付结算，并通过跨境物流送达商品的一种国际商业活动。而26号公告中把跨境电商企业定义为境内注册企业时，消费者与跨境电商企业的交易已经不符合跨境电子商务定义，此次《通知》修改跨境电商企业为境外注册企业，重新实现了消费者与跨境电商企业真正的跨境交易。

四、《通知》第四条第三款第一项“物流企业应取得国家邮政局颁发的《快递业务经营许可证》。”

此项彻底把跨境电子商务物流企业与海关传统物流企业区分开来，是对26号公告的补充。

物流企业(英语:Logistics business)是一种运输、货运、物流公司的类型，全国泛指经营物流相关的运输、仓储、配送、域名注册查询等行业的公司，物流公司常在供货商与零售业者之间扮演集货、理货、库存、配送等角色。所以物流公司有时也会兼营大盘商的角色，物流公司对于供货商而言;可以降低运输与仓储的成本，商品可以直接寄放于物流中心的仓库去出货。不必自己维持一个庞大的仓库去堆积货品，也不

必自己维持货运与配送的庞大车队，这对于中小型的供货商而言是有利的。

由于26号公告中未对跨境电子商务物流企业做具体定义，导致很多海关传统仓储企业也登记成为跨境电子商务的物流企业，这些仓储企业发送物流信息，但却无法实现配送的“最后一公里”，只能和快递企业合作，而由于快递企业未在跨境电子商务系统登记，导致监管部门整个贸易链条“最后一公里”监管的缺失，也容易产生刷单、集中收货等违规现象。此次《通知》明确把快递企业定义为物流企业，是对监管链条的完善，把权责更加明确，有力于跨境电子商务行业的正规发展。

五、《通知》第四条第五款第二项“原则上不允许网购保税进口商品在海关特殊监管区域外开展“网购保税+线下自提”模式。”

此项规定为了保护一般贸易和贸易公平性而制定，消除了一般贸易中保税展示交易货物和跨境电商商品相混淆的现象。由于没有文件规定跨境电商商品可以交纳保证金出区。2017年上半年深圳海关全国首创开展了“保税展示+实体新零售”项目，项目的本质是为了增加消费者体验度，把跨境电子商务商品通过海关特殊区域保税账册互转功能，改变货物属性，以保税展示交易业务交纳保证金出区，消费者可以看到实货样品，决定购买后，消费者在跨境电子商务平台下单，企业向海关系统申报三单信息及申报清单数据，跨境电商商品在海关特殊区域打包，海关系统放行后发货，实现物流配送。如果消费者想购买保税展示的样品，需要按照保税展示交易的相关规定办理报关手续。但后来某些城市却发展成直接按照跨境电商的规定购买保税展示交易货物，违背了项目的初衷。值得注意的是保税展示交易规定虽未提及一般贸易及跨境电子商务字样，但“海关办理保税展示交易货物内销进口征税手续时，按照《中华人民共和国海关审定内销保税货物完税价格办法》（海关总署令第211号）有关规定，以内销价格为基础审定完税价格。”、“对保税展示交易货物内销涉及分批适用协定税率或者特惠税率的，按照海关总署公告2013年第36号有关规定办理。”及“保税展示交易涉及许可证件的，展示经营企业应当在货物销售前向主管海关提供许可证件。”的规定已经明确保税展示交易是一般贸易的业务，由于保税展示交易规定为特别规定，按照《立法法》“特别规定优于一般规定”的原则，即使跨境电子商务商品想开展保税展示交易业务，也必须遵守保税展示交易的规定，将保税展示交易的货物销售给消费者的，涉及许可证件的必须提供许可证件，这是很多跨境电子商务企业无法做到的。综上所述，此项规定彻底截断了批着跨境电子商务的外衣逃避国家监管的行为。

以上都是个人对《通知》的理解，不对的地方请各专家指正。
