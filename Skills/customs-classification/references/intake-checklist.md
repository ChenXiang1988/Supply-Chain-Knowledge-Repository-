# 用户资料清单

## 必填资料
- 商品名称
- 商品用途 / 功能
- 商品材质或主要成分
- 商品结构 / 形态
- 是否成品、半成品、未组装品
- 是否单件商品、套装、组合件、配件包
- 商品照片
- 规格参数
- 包装方式
- 适用的海关体系或申报国家 / 地区
- 如果已有初步税号，提供现有归类思路或候选税号

## 建议补充资料
- 产品说明书
- 技术规格书
- 目录页 / 彩页
- BOM 清单
- 组装图 / 爆炸图
- 成分配比表
- MSDS / 安全数据表
- 检测报告
- 发票或采购清单
- 申报品名
- 品牌、型号、系列
- 销售页面链接
- 实物视频

## 按品类追加提交
### 机电设备
- 工作原理
- 是否带电
- 功率、电压
- 控制方式

### 化工产品
- 化学名称
- CAS 号
- 纯度
- 浓度
- 载体 / 溶剂

### 纺织品
- 纤维成分
- 克重
- 织造方式
- 是否针织 / 梭织

### 食品 / 饮料
- 配方
- 含糖量
- 酒精度
- 是否预包装

### 医疗器械
- 用途
- 注册分类
- 是否用于诊断、治疗、消毒

### 套装商品
- 每个组成部分的明细
- 是否零售包装
- 主导功能是谁

### 包装物
- 是否专用
- 是否可重复使用
- 是否与商品成套销售

## 缺资料时的追问优先级
1. 用途 / 功能
2. 材质 / 成分
3. 是否成套 / 是否未组装
4. 结构 / 规格
5. 包装方式
6. 图片或说明书
