# wx2md - 微信文章转 Markdown 工具

## 快速开始

### 1. 安装依赖（首次使用）

```bash
cd tools/wx2md

# 安装 Python 依赖
pip install -r requirements.txt

# 安装 Playwright 浏览器（只需一次）
playwright install chromium
```

### 2. 粘贴文章链接

打开 `links.txt`，把微信文章链接贴进去，**每行一个**；支持 `#` 开头注释行：

```
https://mp.weixin.qq.com/s/xxxxxxxxx
https://mp.weixin.qq.com/s/yyyyyyyyy

# 这是注释，会被忽略
# https://mp.weixin.qq.com/s/zzzzzzzzz
```

> 获取链接方式：PC 微信打开文章 → 右上角菜单 → 复制链接地址

### 3. 运行脚本

```bash
python wx2md.py
```

> Windows 环境也可使用 `py wx2md.py`

### 4. 查看结果

生成的 Markdown 文件保存在仓库根目录下的 `knowledge_base/web_collected_data/` 目录下：

```
knowledge_base/web_collected_data/
├── 2026-06-14-文章标题A/
│   ├── article.md      ← 正文（Markdown 格式）
│   └── images/         ← 文章中的图片
│       ├── img_001.png
│       └── img_002.jpg
├── 2026-06-14-文章标题B/
│   ├── article.md
│   └── images/
```

每篇文章一个独立文件夹，图片本地化保存，Markdown 中用相对路径引用。

---

## 文章元数据

每个 `article.md` 顶部包含 YAML frontmatter，方便后续检索：

```yaml
---
title: 文章标题
author: 公众号名称
date: 2026-06-14
source_url: https://mp.weixin.qq.com/s/xxxxx
capture_time: 2026-06-14 00:35:00
---
```

---

## 特性说明

### 断点续跑
脚本会自动记录已处理的链接到 `processed_links.txt`，下次运行时会**自动跳过**，无需手动清理 `links.txt`。

### 广告过滤
自动识别并移除文末常见的引流内容，包括：进群、加群、扫码关注、关注公众号、添加微信、私信回复、领取福利等。

---

## 注意事项

- 仅支持 `mp.weixin.qq.com` 域名的文章链接，非微信链接会被自动跳过
- 脚本使用无头浏览器（headless），不会弹出窗口
- 如果抓取失败，可能是微信临时风控，稍后重试即可
- `links.txt` 中的链接处理完后不会自动清空，方便你核对
- `processed_links.txt` 保存已处理记录，如需重新抓取某篇文章，先从该文件中删除对应链接



# 小白版本
# wx2md - 微信文章转 Markdown 工具
## 前置重要说明（所有人必看）
本工具为完整文件夹分发，**直接复制整个文件夹发给对方即可使用**，无需额外下载软件。
使用全程跟着下面步骤一步步操作，看不懂命令也没关系，每一步都写得非常详细。

### 一、使用前强制检查（避免90%报错）
1. 把收到的整个文件夹，**移动/复制到纯英文路径下**
   ✅ 正确示例：`D:\job\web_collected_data\wx2md`
   ❌ 错误示例（绝对不要用）：`D:\微信工具\转md`、`D:\我的文件\wx 2md`
   规则：路径里**不能出现中文、汉字、空格、特殊符号**。
2. 确保电脑**网络正常**，工具需要联网加载组件、访问微信文章。
3. 全程操作时，**不要删除、重命名**文件夹内任何原有文件。

---

## 快速开始
### 1. 安装依赖（首次使用，只需要做这一次）
> 说明：首次使用必须执行下面命令，后续再次使用可直接跳过本步骤。

#### 1.1 打开终端窗口（PowerShell / 命令提示符）
1. 打开你存放 `wx2md` 的文件夹
2. 在文件夹**顶部地址栏**点击一下，全选原有地址，输入 `cmd`，按下回车键
   此时会弹出一个黑色窗口（命令行窗口），当前路径已经自动定位到本文件夹。

#### 1.2 依次执行下方两条命令
在黑色窗口里，**复制一行、粘贴一行、按回车**执行：
```bash
cd tools/wx2md
# 安装 Python 依赖
pip install -r requirements.txt
# 安装 Playwright 浏览器（只需一次）
playwright install chromium
```
- 粘贴后按回车，等待文字滚动结束，出现新的输入行，代表当前命令执行完成。
- 全程等待即可，不要中途关闭黑色窗口。

### 2. 粘贴文章链接
1. 在文件夹里找到名为 `links.txt` 的文件，**双击用记事本打开**。
2. 把微信文章链接贴进去，**每行只放一个链接**；以 `#` 开头的行是注释，程序不会处理。

填写示例：
```
https://mp.weixin.qq.com/s/xxxxxxxxx
https://mp.weixin.qq.com/s/yyyyyyyyy
# 这是注释，会被忽略
# https://mp.weixin.qq.com/s/zzzzzzzzz
```

#### 👉 小白专属：如何获取微信文章链接？
1. 打开电脑版微信，点开需要转换的公众号文章。
2. 点击文章页面**右上角「···」更多按钮**。
3. 在弹出菜单里选择 **复制链接地址**。
4. 回到 `links.txt`，右键粘贴链接即可。
5. 全部链接粘贴完成后，**按下 Ctrl + S 保存文件**，关闭记事本。

### 3. 运行脚本
依旧保持之前打开的**黑色命令行窗口**（如果关闭了，重新按 1.1 步骤打开）。

在窗口中输入下面命令，输入完成后按下回车键：
```bash
python wx2md.py
```
> Windows 环境也可使用 `py wx2md.py`

#### 运行期间注意事项
1. 命令执行后，窗口会不断弹出文字，代表正在处理文章。
2. **运行过程中绝对不要关闭黑色窗口**，耐心等待。
3. 看到最后一行提示「全部任务执行完毕」，代表运行成功。

### 4. 查看结果
生成的 Markdown 文件保存在仓库根目录下的 `knowledge_base/web_collected_data/` 目录下：
```
knowledge_base/web_collected_data/
├── 2026-06-14-文章标题A/
│   ├── article.md      ← 正文（Markdown 格式）
│   └── images/         ← 文章中的图片
│       ├── img_001.png
│       └── img_002.jpg
├── 2026-06-14-文章标题B/
│   ├── article.md
│   └── images/
```
每篇文章一个独立文件夹，图片本地化保存，Markdown 中用相对路径引用。

#### 打开文件说明
1. 找到对应文件夹里的 `article.md`，推荐使用 Typora、Obsidian 等专业 Markdown 软件打开。
2. **不要使用系统自带记事本打开**，记事本无法正常显示图片。
3. 禁止单独移动 `article.md` 文件，必须和同目录下的 `images` 文件夹放在一起，否则图片会丢失。

## 🆘 常见问题 & 故障排查（遇到报错优先看这里）
### 问题1：输入命令后提示 `python 不是内部或外部命令`
解决：
1. 代表本机未配置好 Python 环境，尝试替换命令为：
   ```bash
   py wx2md.py
   ```
2. 若依旧报错，检查文件夹路径是否有中文/空格，重新移动到纯英文路径再试。

### 问题2：运行命令后窗口一闪而过，没有任何反应
解决：
1. 检查文件夹路径，**必须纯英文、无空格、无中文**。
2. 打开 `links.txt`，确认里面有**有效的微信文章链接**，不是空文件。
3. 重新按步骤打开命令行窗口，再执行命令。

### 问题3：转换后的 `article.md` 打开看不到图片
解决：
1. 确认 `article.md` 和 `images` 文件夹在**同一个文件夹内**，没有拆分、移动。
2. 使用 Typora / Obsidian 等 Markdown 专用软件打开，记事本不支持图片预览。

### 问题4：提示链接处理失败、抓取失败
解决：
1. 检查网络是否正常，断开重连网络后重试。
2. 核对链接：必须是完整微信文章链接，小程序链接、短链接、朋友圈链接都无法使用。
3. 微信有临时访问限制，等待 5~10 分钟后重新运行脚本。

### 问题5：想重新转换已经处理过的旧文章
解决：
1. 打开文件夹内的 `processed_links.txt`。
2. 删除里面对应的那条微信链接，保存文件。
3. 重新执行运行命令即可重新抓取。

### 问题6：首次安装依赖时报错、下载卡住
解决：
1. 检查网络，切换手机热点/稳定网络后重新执行安装命令。
2. 不要中途关闭窗口，网络较慢时下载会耗时较久。
